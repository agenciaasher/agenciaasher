# Agvel Veículos — Identidade Visual

## Logo

4 versões disponíveis nesta pasta:

| Arquivo | Uso |
|---------|-----|
| `logo-cromado.png` | Versão principal. Efeito metálico/cromado. Usar em fundo branco ou claro. |
| `logo-preto.png` | Versão sólida preta. Landing pages, documentos, fundos claros. |
| `logo-branco-fundo-escuro.png` | Versão branca. Usar em fundos escuros/pretos. |
| `simbolo-preto.png` | Só o "A". Favicon, ícones, marca d'água, redes sociais. |

Arquivos editáveis (.AI) ficam no Drive do Wenderson.

## Tipografia do Logo

- Estilo: bold, itálico/inclinado, sem serifa
- Referência visual: automotivo, velocidade, robustez
- Typeface customizada (não é fonte padrão)

## Paleta de Cores

Extraída do brandbook:

| Cor | Hex | Uso |
|-----|-----|-----|
| **Preto** | `#1D1D1B` | Cor principal. Textos, fundos escuros, contornos do logo. |
| **Branco** | `#FFFFFF` | Fundos claros, textos sobre fundo escuro. |
| **Cinza escuro** | `#333333` | Textos secundários, subtítulos. |
| **Prata/Cromado** | Gradiente `#C0C0C0` → `#FFFFFF` → `#808080` | Efeito metálico do logo principal. Usar com moderação. |

### Cores complementares (sugestão para digital)

| Cor | Hex | Uso |
|-----|-----|-----|
| **Vermelho escuro** | `#CC0000` | CTAs, botões de ação, destaques de preço. Usar com parcimônia. |
| **Cinza claro** | `#F5F5F5` | Fundos de seção, backgrounds de cards. |

## Regras de Uso

1. O logo cromado é a versão principal — usar sempre que possível.
2. Nunca distorcer, rotacionar além do itálico original, ou alterar cores do logo.
3. Manter área de respiro ao redor do logo (mínimo 1x a altura do "A").
4. Em fundos escuros, usar SEMPRE a versão branca. Nunca colocar logo preto em fundo escuro.
5. O tom visual é sério e premium — combina com as referências (ShiftCar, TCar, Avantgarden).
6. Para landing pages: usar `logo-preto.png` no header e `simbolo-preto.png` como favicon.
