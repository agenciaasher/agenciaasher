## Fluxo de Trabalho

### Como funciona

1. **Maicon** define a estrategia e as prioridades do mes
2. **Wiliam** recebe, distribui as tarefas para o time e cobra prazos
3. Cada membro executa sua parte
4. **Wiliam** faz o CS com o cliente (reunioes, alinhamentos, entregas)
5. **Wiliam** reporta para Maicon o status dos projetos

### Fluxo semanal

```
Segunda     → Wiliam alinha prioridades da semana com o time
Diariamente → Time executa, Wiliam acompanha
Sexta       → Wiliam consolida status e reporta para Maicon
Mensal      → Reuniao de resultado com cada cliente (Wiliam + Maicon)
```

### Fluxo de Copy (via IA / Skills)

Como a copy e feita via IA com skills do Claude Code:

1. Maicon ou Wiliam roda a skill com o briefing do cliente
2. IA gera a copy (anuncio, landing page, script, objecoes)
3. Maicon revisa e aprova
4. Wiliam distribui para o time (Wenderson para design, Julio para trafego)

### Fluxo por entregavel

**Trafego Pago (Julio Cesar)**
- Recebe: copy aprovada + criativos do Wenderson
- Executa: sobe campanhas, otimiza semanalmente, gera relatorio
- Reporta: para Wiliam (que repassa ao cliente)

**Design e Video (Wenderson)**
- Recebe: briefing + copy aprovada
- Executa: criativos de anuncio, landing page, reels, thumbnails, edicao de video
- Entrega: para Julio (trafego) e Annick (social media)

**Social Media (Annick)**
- Recebe: pauta mensal + criativos do Wenderson
- Executa: publicacoes, stories, gestao de comunidade
- Apoio: inteligencia de conteudo (temas, benchmarks, insights)

**Coordenacao (Wiliam)**
- Distribui todas as tarefas vindas de Maicon
- Acompanha prazos e qualidade
- Faz CS com os clientes (reunioes, alinhamentos)
- Gerencia o setor comercial digital do cliente (scripts, funil, follow-up)
- Reporta status para Maicon
