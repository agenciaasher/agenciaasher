Como Usar as Skills

Passo a passo para qualquer pessoa do time

1

Abra o Claude Code

No terminal, navegue até a pasta do projeto da agência e digite claude para iniciar.

cd agenciaasher
claude
2

Chame a skill pelo nome

Digite o nome da skill que quer usar. O Claude vai reconhecer automaticamente e seguir o fluxo.

Roda a skill de pesquisa de mercado para a Agvel

Usa a skill de mapa de objeções para o Clube Clincog

Gera um criativo surreal para a campanha de SUVs da Agvel
3

Responda as perguntas

Cada skill vai pedir informações específicas (nicho, produto, público, etc.). Responda com os dados do cliente. Se for cliente da agência, a skill já puxa dados do briefing em clientes/<nome>/projeto.md.

4

Aprove ou ajuste

A maioria das skills pede aprovação bloco a bloco. Se algo não ficou bom, peça ajuste. O Claude regera só aquele trecho sem refazer o resto.

1. Aprovar e seguir
2. Quero ajustar (diga o que mudar)
5

Receba o entregável

No final, a skill entrega o material pronto: copy formatada, relatório, briefing de design, prompts para ChatGPT, etc. Copie, baixe ou passe para o próximo setor.

Dica: As skills funcionam melhor quando você menciona o cliente. Exemplo: "Roda a skill de concepcao de produto para o Clube Clincog, assinatura mensal de R$97". O Claude vai buscar os dados do cliente automaticamente e adaptar tudo pro contexto real.
