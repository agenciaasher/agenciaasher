Como Usar as Skills

Passo a passo para qualquer pessoa do time

1

Abra o Claude Code

No terminal, navegue ate a pasta do projeto da agencia e digite claude para iniciar.

cd agenciaasher
claude
2

Chame a skill pelo nome

Digite o nome da skill que quer usar. O Claude vai reconhecer automaticamente e seguir o fluxo.

Roda a skill de pesquisa de mercado para a Agvel

Gera o script de whatsapp para a Clincog, produto Clube

Cria a pauta mensal de julho para a Agvel

Analisa essas metricas da campanha da Clincog
3

Responda as perguntas

Cada skill vai pedir informacoes especificas (nicho, produto, publico, etc.). Responda com os dados do cliente. Se for cliente da agencia, a skill ja puxa dados do briefing em clientes/<nome>/projeto.md e da sazonalidade em clientes/<nome>/sazonalidade.md.

4

Aprove ou ajuste

A maioria das skills pede aprovacao bloco a bloco. Se algo nao ficou bom, peca ajuste. O Claude regera so aquele trecho sem refazer o resto.

1. Aprovar e seguir
2. Quero ajustar (diga o que mudar)
5

Receba o entregavel

No final, a skill entrega o material pronto: copy formatada, relatorio, briefing de design, roteiro de reels, landing page publicada, diagnostico de campanha. Copie, baixe ou passe para o proximo setor.

Dica: As skills funcionam melhor quando voce menciona o cliente. Exemplo: "Roda a skill de copy de anuncio para a Agvel, campanha de SUVs, fundo de funil". O Claude vai buscar os dados do cliente, a sazonalidade do mes e os benchmarks de CPA automaticamente.
